# DP Forensic Psychology — Simple Website

This is a minimal static website for DP Forensic Psychology. It includes a main page with a dropdown menu linking to `CV`, `Services`, and `Contact`.

How to use

Open `index.html` in your browser. No build step required.

Files

- index.html — main page with hero, about, services, contact sections
- cv.html — simple CV page
- styles.css — site styles
- script.js — dropdown + form stub behavior
